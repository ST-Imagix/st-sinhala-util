# 🇱🇰 st-sinhala-util

The Ultimate Sinhala Language Toolkit for Developers. A lightweight, high-performance JavaScript utility library designed specifically for Sri Lankan software developers. 🚀

🔗 **Try Demo:** [https://st-imagix.github.io/st-sinhala-util/](https://st-imagix.github.io/st-sinhala-util/)

---

## ✨ Features (v1.0.0)
- **Number to Words:** Convert any number into grammatically correct Sinhala text.
- **Billing Ready:** Perfect for invoice generation, banking apps, and receipt printing.
- **Fast & Light:** Zero dependencies.


## 🚀 How to Use .

JavaScript
const { numberToSinhala } = require('st-sinhala-util');

console.log(numberToSinhala(1250)); 
// Output: එක්දහස් දෙසිය පනහයි
