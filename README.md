# st-sinhala-util
A lightweight JavaScript utility library for Sinhala text processing, including number-to-words conversion, date formatting, and Unicode validation.
# 🇱🇰 The Ultimate Sinhala Language Toolkit for Developers

A lightweight, high-performance JavaScript utility library designed specifically for Sri Lankan software developers. 🚀

## ✨ Features (v1.0.0)
- **Number to Words:** Convert any number into grammatically correct Sinhala text.
- **Billing Ready:** Perfect for invoice generation, banking apps, and receipt printing.
- **Fast & Light:** Zero dependencies.
